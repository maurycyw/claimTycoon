# Claim Tycoon

A tycoon simulation game built with Unity.

## Project Structure

- **Assets/**: Unity project assets including scripts, scenes, and settings.
- **Planning/**: Design documents and project plans.
- **Packages/**: Unity package manager manifests.
- **ProjectSettings/**: Unity project settings.

## Getting Started

1. Open the project in Unity (refer to `ProjectSettings/ProjectVersion.txt` for the specific Unity version).
2. Open `Assets/Scenes/SampleScene.unity` to see the initial scene.
